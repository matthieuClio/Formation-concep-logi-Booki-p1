# Booki – Responsive website integration (HTML / CSS)

## Project context
This project was developed as part of the **Application Designer & Developer (React-oriented)** training program by OpenClassrooms.  
It's a project of the program and focuses on front-end integration fundamentals before introducing JavaScript and React.

The objective was to integrate a responsive accommodation booking website based on a provided design, following strict layout and responsiveness requirements.

## Educational objectives
- Integrate a complete website using HTML and CSS
- Apply semantic HTML structure
- Build a fully responsive layout (mobile, tablet, desktop)
- Use modern CSS techniques (Flexbox)
- Respect a design mockup and technical constraints
- Organize front-end assets in a clean project structure

## Technologies used
- HTML5
- CSS3
- Flexbox
- Git / GitHub
- GitHub Pages (deployment)

## Features
- Homepage displaying accommodation listings
- Popular section with highlighted content
- Filters section for accommodations
- Responsive layout adapting to different screen sizes
- Pixel-close integration based on provided mockups
